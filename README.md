# 🌍 Overview

This project showcases a **professional weather analytics dashboard** built in **Power BI** using a **live Weather API**. It demonstrates API integration, data transformation with Power Query, DAX calculations, and modern dashboard design.

---

# ✨ Features

- 🌡️ Current Temperature
- 💧 Humidity
- 🌬️ Wind Speed
- 🌧️ Rain Probability
- ☀️ UV Index
- 🌅 Sunrise & Sunset
- 👀 Visibility
- 📍 Live Weather Conditions
- 📈 Interactive KPIs
- 🔄 Refreshable API Data
